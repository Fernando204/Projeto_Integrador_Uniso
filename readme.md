# 🚀 MicroGestor

**A plataforma inteligente para gestão completa de micro e pequenas empresas.**

O **MicroGestor** é uma aplicação desenvolvida para centralizar, organizar e simplificar a administração de negócios de pequeno porte. A solução foi pensada para resolver problemas reais do dia a dia empresarial, oferecendo controle, clareza e eficiência em um único sistema.

---

## 🧠 Visão do Produto

Pequenos negócios costumam depender de planilhas, anotações manuais e controles descentralizados. O **MicroGestor** nasce com o objetivo de substituir esse cenário por uma plataforma moderna, intuitiva e confiável.

Mais do que um sistema de controle, o MicroGestor atua como um **apoio estratégico** para o empreendedor, ajudando na tomada de decisões com base em dados claros e organizados.

---

## 🎯 Principais Funcionalidades

- 📊 Painel de controle (Dashboard)
- 📦 Gestão de produtos e estoque
- 🧾 Registro e acompanhamento de movimentações
- 👥 Cadastro e gerenciamento de usuários e empresas
- 💰 Controle financeiro básico
- 📑 Relatórios gerenciais
- 🔐 Autenticação e segurança
- 📱 Interface responsiva

## 🚀 Como Executar o Projeto

### 1. Clone o repositório
```bash
git clone https://github.com/Fernando204/Projeto_Integrador_Uniso.git
```

### 2. Acesse o diretório
```bash
cd Projeto_Integrador_Uniso
```

### 3. Configure o banco de dados
- Crie um banco MySQL
- Ajuste as configurações no arquivo `application.properties`

### 4. Execute o backend
```bash
mvn spring-boot:run
```

### 5. Acesse o sistema
Abra o frontend no navegador ou utilize as rotas da API.
