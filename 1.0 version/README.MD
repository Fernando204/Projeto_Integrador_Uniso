inicio: 19/02/2026

O projeto é constituido de dois principais diretórios, um para o back-end e outro para o front-end


# Front-end:

*O front-end é separado em diversas pastas e arquivos sendo eles:

 - components: componentes do html que serão carregados dinamicamente no sistema principal
    incluindo as pastas separando o componente por categoria além de incuir os arquivos html, css e javascrip de cada componente

 - images: imagens do sitema como a logo

 - Pages: paginas html completas que tem como fim possibilitar o usuário realzar ações antes de entrar no sistema
    como login, registro e até mesmo consultar informações

 - scripts: arquivos de código javascript que serão usados na plataforma de modo geral como a classe de conexão com a api
    configuração de rotas e o arquivo principal da mainpage do sistema

 - styles: pastas dos arquivos css que serão usados de modo geral nas paginas

 - index.html: é o arquivo principal do sistema onde serão carregados os componentes das ferramentas
